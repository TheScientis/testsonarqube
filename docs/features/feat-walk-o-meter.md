# Feature: Walk-o-Meter — Map & Evidence Feed

> **File naming:** `feat-walk-o-meter.md`

---

## 1. Overview

| Field | Description |
|-------|-------------|
| **Feature ID** | `F-003` |
| **Objective** | To provide a public map and evidence feed that visualizes environmental accountability: where promises meet reality. |
| **Summary** | The Walk-o-Meter map shows evidence pins (e.g. Surat Pengaduan shared from Bang Jaga, community reports) and supplies a **Walk-o-Meter score** per promise (used in The Talk Ledger). Score combines community Yes/No votes (60%) with official progress (40%). |
| **Related PRD** | PRD §4.3 |
